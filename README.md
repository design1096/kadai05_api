# ①課題番号-プロダクト名	
- JS5 - APIを使ったアプリ
- タイトル：NASA's Photo

## ②課題内容（どんな作品か）
- NASAのAPIを使用して画像を検索、表示できるアプリ
- 「NASA's Daily Photo」ボタン押下で本日の一枚を表示
- 「日本語」ボタン押下で日本語翻訳文を表示（翻訳API利用）
- キーワードを入力して「Search Photos」ボタン押下でキーワードに当てはまる画像を表示
- 画像をクリックすることで拡大表示
- 「×」ボタン押下で画像表示エリアが非表示になる
- ※DEMO版では翻訳APIのURLを一部非公開にしているため、「日本語」ボタンが機能しません※
